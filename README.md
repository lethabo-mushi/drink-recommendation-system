# Drink Recommendation System 🍹

## Overview
This project is a simple recommendation system that suggests drinks based on user preferences such as flavor, sweetness, and alcohol content.

## Features
- User input-based recommendations
- Data filtering using Pandas
- Fallback suggestions when no exact match is found

## Technologies Used
- Python
- Pandas

## Purpose
This project demonstrates basic understanding of data analysis, user preference modeling, and recommendation systems.
